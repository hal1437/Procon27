
## 環境
* C++11 
* openCV 2.4.12 
* cmake 3.5.1
* Apple LLVM version 6.0

## 導入方法
gitのリモートリポジトリからクローンし、作業ディレクトリへ移動
```
git clone https://github.com/hal1437/Procon27
cd Procon27
```

ビルドディレクトリを作成、移動
```
mkdir build
cd build
```

ビルド
```
cmake ../solver
make
```

実行
```
bin/master
```
